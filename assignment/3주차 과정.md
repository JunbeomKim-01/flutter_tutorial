## :clipboard: Assignment

- Map을 이용한 자료형을 가진 임의의 데이터를 생성하고, 해당 데이터를 화면에 출력합니다. (예시 스크린은 두 개의 데이터를 반복) 
- 임의의 데이터에 날짜를 포함하고 해당 데이터를 가공하여 D-Day와 날짜를 예시 스크린의 포맷으로 출력 합니다.
